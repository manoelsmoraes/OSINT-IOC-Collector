# 🔎 OSINT IOC Collector

Projeto desenvolvido para coleta e análise de indicadores de comprometimento (IOCs) a partir de fontes abertas (OSINT).

## 🎯 Objetivo
Demonstrar na prática a aplicação de técnicas de Threat Intelligence para:
- Coleta de dados
- Extração de IOCs
- Apoio a operações de SOC

## ⚙️ Funcionalidades
- Coleta de dados de fonte OSINT pública
- Extração de IPs e domínios
- Exibição de indicadores relevantes

## 🛠️ Tecnologias
- Python
- Requests
- Regex

## 🚀 Como executar

```bash
pip install -r requirements.txt
python main.py
